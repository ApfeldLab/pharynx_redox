# Worm Analysis

## Getting the Data
You'll need access to the shared folder for the training data etc.. It's all in
```
Sean > WormAnalysis > data
```

Copy that directory (or symlink it) to ./data

## Environment setup
running `pip install -r requirements.txt` should set you up
